Detailed Description

template<typename Tile, typename Policy>
class mpqc::scf::FourCenterFockBuilder< Tile, Policy >

FourCenterFockBuilder is an integral-direct implementation of FockBuilder in a Gaussian AO basis that uses 4-center integrals and optimally takes advantage of the permutational symmetry and shell-level screening.
