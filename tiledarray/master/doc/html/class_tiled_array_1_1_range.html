Detailed Description

A (hyperrectangular) interval on , space of integer n-indices.

This object represents an n-dimensional, hyperrectangular array of integers. It provides information on the rank (number of dimensions), (nonnegative) lower bound, upper bound, extent (size), and stride of each dimension. It can also be used to test if an element is included in the range with a coordinate index or ordinal offset. Finally, it can be used to convert coordinate indices to ordinal offsets and vice versa.
