Detailed Description

template<typename Value, typename Container>
class TiledArray::detail::RangeIterator< Value, Container >

Coordinate index iterate.

This is an input iterator that is used to iterate over the coordinate indexes of a Range.

Template Parameters
ValueThe value type of the iterator
ContainerThe container that the iterator references
Note
The container object must define the function Container::increment(Value&) const, and be accessible to RangeIterator.
