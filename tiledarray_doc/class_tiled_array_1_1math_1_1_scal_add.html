Detailed Description

template<typename Result, typename Left, typename Right, bool LeftConsumable, bool RightConsumable>
class TiledArray::math::ScalAdd< Result, Left, Right, LeftConsumable, RightConsumable >

Tile addition and scale operation.

This addition operation will add the content two tiles, then scale and apply a permutation to the result tensor. If no permutation is given or the permutation is null, then the result is not permuted.

Template Parameters
ResultThe result type
LeftThe left-hand argument type
RightThe right-hand argument type
LeftConsumableA flag that is true when the left-hand argument is consumable.
RightConsumableA flag that is true when the right-hand argument is consumable.
