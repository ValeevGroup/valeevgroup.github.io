Detailed Description

template<typename Tile, typename Policy, typename Factory>
class mpqc::scf::PeriodicMARIJCADFKFockBuilder< Tile, Policy, Factory >

PeriodicMARIJCADFKFockBuilder is an implementation of PeriodicFockBuilder with multipole-accelerated RI-J and CADF exchange. For Coulomb, multipole approximation is used in Crystal Far Field and RI-J in Crystal Near Field. For exchange, CADF-K is always used.
