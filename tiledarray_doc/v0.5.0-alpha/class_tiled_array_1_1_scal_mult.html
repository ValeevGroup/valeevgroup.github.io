Detailed Description

template<typename Left, typename Right, typename Scalar, bool LeftConsumable, bool RightConsumable>
class TiledArray::ScalMult< Left, Right, Scalar, LeftConsumable, RightConsumable >

Tile scale-multiplication operation.

This multiplication operation will multiply the content two tiles and apply a permutation to the result tensor. If no permutation is given or the permutation is null, then the result is not permuted.

Template Parameters
LeftThe left-hand argument type
RightThe right-hand argument type
ScalarThe scaling factor type
LeftConsumableA flag that is true when the left-hand argument is consumable.
RightConsumableA flag that is true when the right-hand argument is consumable.
