Detailed Description

template<typename Left, typename Right, bool LeftConsumable, bool RightConsumable>
class TiledArray::Add< Left, Right, LeftConsumable, RightConsumable >

Tile addition operation.

This addition operation will add the content two tiles, and accepts an optional permute argument.

Template Parameters
LeftThe left-hand argument type
RightThe right-hand argument type
LeftConsumableA flag that is true when the left-hand argument is consumable.
RightConsumableA flag that is true when the right-hand argument is consumable.
