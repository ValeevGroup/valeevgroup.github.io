Detailed Description

template<typename T, unsigned int DIM, typename Tile = Tensor<T, Eigen::aligned_allocator<T> >, typename Policy = DensePolicy>
class TiledArray::Array< T, DIM, Tile, Policy >

An n-dimensional, tiled array.

Forward declarations.

Array is the local representation of a global object. This means that the local array object will only contain a portion of the data. It may be used to construct distributed tensor algebraic operations.

Template Parameters
TThe element type of for array tiles
DIMThe number of dimensions for this array object
TileThe tile type [ Default = Tensor<T> ]

Initialize tiles with a user provided functor.

This function is used to initialize tiles of the array via a function (or functor). The work is done in parallel, therefore op must be a thread safe function/functor. The signature of the functor should be:
