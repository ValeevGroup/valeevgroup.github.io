◆ Formula() [4/4]

mpqc::Formula::Formula ( std::wstring  formula)

Constructor parses the string in one of the following formats:

  • Physical Notation <index1 index2|operator|index3 index4>[option1,option2]
  • Chemical Notation (index1 index2|operator|index3 index4)[option]

where the index keys index1 , , etc. are parsed by mpqc::lcao::OrbitalIndex, the operator key operator is parsed by Operator , and the option keys can be one of the following:

  • df -> Option::DensityFitting
  • inv -> Option::Inverse
  • inv_sqr -> Option::InverseSquareRoot
  • <symm> -> math::PetiteList::Symmetry
