45  // Type traits for detecting tensors and tensors of tensors.
46  // is_tensor_helper tests if individual types are tensors, while is_tensor
47  // tests zero or more tensor types. Similarly is_tensor_of_tensor tests if
48  // one or more types are tensors of tensors.
49  // To extend the definition of tensors and tensors of tensor, add additional
50  // is_tensor_helper and is_tensor_of_tensor_helper (partial) specializations.
51  // Note: These type traits help differentiate different implementation
52  // functions for tensors, so a tensor of tensors is not considered a tensor.
53

55  template <typename...Ts> struct is_tensor_of_tensor;
