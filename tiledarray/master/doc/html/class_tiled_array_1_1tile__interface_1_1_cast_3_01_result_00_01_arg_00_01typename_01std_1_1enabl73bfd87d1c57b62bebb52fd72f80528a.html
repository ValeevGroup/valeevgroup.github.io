Detailed Description

template<typename Result, typename Arg>
class TiledArray::tile_interface::Cast< Result, Arg, typename std::enable_if< is_lazy_tile< Arg >::value &&! std::is_same< Result, typename TiledArray::eval_trait< Arg >::type >::value >::type >

Internal cast implementation.

This class is used to define internal tile cast operations. This specialization handles casting of lazy tiles to a type other than the evaluation type.

Template Parameters
ResultThe output tile type
ArgThe input tile type
Note
this specialization is invoked when Arg is a lazy tile (see TiledArray::is_lazy_tile) that evaluates to something other than Result.

Tile cast operation.

Cast arg to a result_type tile. This potentially involves 2 steps:

  • from argument_type to eval_t<argument_type>
  • from eval_t<argument_type> to result_tile These casts are nonblocking, if needed.
