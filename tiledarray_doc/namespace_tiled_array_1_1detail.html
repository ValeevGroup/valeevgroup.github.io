template<typename Perm , typename Arg , typename Result >
void TiledArray::detail::permute_array ( const Perm &  perm,
const Arg &  arg,
Result &  result 
)
inline

Copies an iterator range into an array type container.

Permutes iterator range [first_o, last_o) base on the permutation [first_p, last_p) and places it in result. The result object must define operator[](std::size_t).

  • [first_p, last_p) is an iterator range to the permutation
  • [irst_o is an input iterator to the beginning of the original array
  • result is a random access iterator that will contain the resulting permuted array
