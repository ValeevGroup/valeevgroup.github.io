Detailed Description

template<typename Tile, typename Policy>
class mpqc::scf::PeriodicMAFourCenterJCADFKFockBuilder< Tile, Policy >

PeriodicMAFourCenterJCADFKFockBuilder is an implementation of PeriodicFockBuilder with multipole-accelerated 4-center Coulomb and CADF exchange. For Coulomb, multipole approximation is used in Crystal Far Field and 4-center-J in Crystal Near Field. For exchange, CADF-K is always used.
