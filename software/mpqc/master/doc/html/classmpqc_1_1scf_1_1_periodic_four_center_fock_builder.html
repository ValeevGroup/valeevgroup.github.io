Detailed Description

template<typename Tile, typename Policy>
class mpqc::scf::PeriodicFourCenterFockBuilder< Tile, Policy >

PeriodicFourCenterFockBuilder is an integral-direct implementation of PeriodicFockBuilder with periodic Gaussian AO basis that uses 4-center integrals. This builder takes advantage of shell-level screening and parallelized summation over unit cells for coulomb interaction
