Detailed Description

template<typename Result, typename Arg>
class TiledArray::tile_interface::Cast< Result, Arg, std::enable_if_t< detail::has_conversion_operator< Arg, madness::Future< Result > >::value||detail::is_convertible< Arg, Result >::value > >

Internal cast (aka conversion) implementation.

This class is used to define internal tile cast operations. Users may specialize the TiledArray::Cast class.

Template Parameters
ResultThe output tile type
ArgThe input tile type
EnablerEnabler type used to select (partial) specializations
Note
the base implementation is invoked when Arg is a lazy tile (see TiledArray::is_lazy_tile) that evaluates to Result.
