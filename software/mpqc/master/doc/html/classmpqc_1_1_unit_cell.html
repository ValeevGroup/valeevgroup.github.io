Parameters
kvThe KeyVal object will be queried for all keywords of OrbitalBasisRegistry, and the following keywords:
Keyword Type Default Description
lattice_param int 0 This gives unit cell dimensions.

example input:

"unitcell": {
"file_name": "water.xyz",
"sort_input": true,
"lattice_param": [0.0, 0.0, 2.672359],
}
