template<typename Tile, typename Policy>
class mpqc::lcao::gaussian::PeriodicAOFactory< Tile, Policy >

Periodic Integral Class.

This class computes atomic integrals involved in periodic calculations using Formula

compute(formula) returns TArray object

KeyVal constructor for PeriodicAOFactory.

Parameters
kvThe KeyVal object will be queried for all the following keywords:
Keyword Type Default Description
rmax array<int, 3> none This gives range of expansion of Bloch Gaussians in AO Gaussians.
rdmax array<int, 3> none This gives range of Coulomb operation.
rjmax array<int, 3> none This gives range of density representation.
engine_precision real machine epsilon This gives integral engine precision.
screen string schwarz This gives method of screening, qqr or schwarz.
threshold real 1e-20 This gives threshold for schwarz or qqr screening.
shell_pair_threshold real 1e-12 This gives threshold for screeing non-negligible shell pairs.
density_threshold real sparse shape threshold This gives threshold for screening density blocks in Fock build.
print_detail bool false Print more details if true.

example input:

"wfn_world": {
"atoms" : "$:water",
"basis" : "$:basis",
"df_basis" : "$:dfbs",
"screen": "schwarz",
"threshold": 1.0e-20,
"shell_pair_threshold": 1.0e-20,
"density_threshold": 1.0e-10,
"rmax": [0, 0, 10],
"rjmax": [0, 0, 10],
"rdmax": [0, 0, 10]
}
