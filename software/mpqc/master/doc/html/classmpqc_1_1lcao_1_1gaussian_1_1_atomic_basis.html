Detailed Description

AtomicBasis is a Basis constructed from a collection of atoms, i.e. a Molecule( or a UnitCell). Updating the Molecule object updates this basis as well. Clustering of AtomicBasis does not have to correspond to the cluster structure of Molecule (e.g. as a result of reblocking).

Note
Shells are organized in the order of increasing atom index.
