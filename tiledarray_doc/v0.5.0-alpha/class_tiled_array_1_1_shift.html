Detailed Description

template<typename Arg, bool Consumable>
class TiledArray::Shift< Arg, Consumable >

Tile shift operation.

This no operation will shift the range of the tile and/or apply a permutation to the result tensor.

Template Parameters
ResultThe result type
ArgThe argument type
ConsumableFlag that is true when Arg is consumable
