Detailed Description

template<typename T>
struct TiledArray::eval_trait< T, typename std::enable_if< detail::is_type< typename T::eval_type >::value >::type >

Determine the object type used in the evaluation of tensor expressions.

This trait class allows user to specify the object type used in an expression by providing a member type T::eval_type. This allows users to use lazy tile construction inside tensor expressions. If no evaluation type is specified, the lazy tile evaluation is disabled.
