Detailed Description

Range data of an N-dimensional block.

This object represents an N-dimension, orthogonal block of elements. It provides information on the rank (number of dimensions), lower bound, upper bound, extent (size), and stride of block. It can also be used to test if an element is included in the range with a coordinate index or ordinal offset. Finally, it can be used to convert coordinate indices to ordinal offsets and vice versa.
