Detailed Description

template<typename Result, typename Arg, bool Consumable>
class TiledArray::math::Neg< Result, Arg, Consumable >

Tile negation operation.

This negation operation will negate the content a tile and apply a permutation to the result tensor. If no permutation is given or the permutation is null, then the result is not permuted.

Template Parameters
ResultThe result type
ArgThe argument type
ConsumableFlag that is true when Arg is consumable
