Detailed Description

template<typename Left, typename Right, bool LeftConsumable, bool RightConsumable>
class TiledArray::Subt< Left, Right, LeftConsumable, RightConsumable >

Tile subtraction operation.

This subtraction operation will subtract the content two tiles, and accepts an optional permute argument.

Template Parameters
LeftThe left-hand argument base type
RightThe right-hand argument base type
LeftConsumableA flag that is true when the left-hand argument is consumable.
RightConsumableA flag that is true when the right-hand argument is consumable.
