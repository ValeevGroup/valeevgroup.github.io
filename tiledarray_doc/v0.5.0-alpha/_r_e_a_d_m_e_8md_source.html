3 ### Synopsis
4 TiledArray is a scalable, block-sparse tensor library that is designed to aid in rapid composition of high-performance tensor expressions, appearing for example in many-body quantum mechanics. It allows users to compose tensor expressions of arbitrary complexity in native C++ code that closely resembles the standard mathematical notation. The library is designed to scale from a single multicore computer to a massive cluster.
5 
6 TiledArray is built on top of MADNESS parallel runtime (MADWorld), part of [MADNESS numerical calculus framework](https://github.com/m-a-d-n-e-s-s/madness).
7 
8 TiledArray is a work in progress.
9 
10 ### Design Goals
11 * Simple implementation of tensor algebra in C++
12 * Dense, and structured/unstructured sparse tensors
13 * Intra- and inter-node scalability
14 * General purpose
15 
16 ### Example Code
17 The following are example expressions written in C++ with TiledArray. TiledArray use the [Einstein summation convention](http://en.wikipedia.org/wiki/Einstein_notation) when evaluating tensor expressions.
18 
19 * Matrix-matrix multiplication
20 
21  C("m,n") = 2.0 * A("m,k") * B("k,n");
22 
23 * Matrix-vector multiplication
24 
25  C("n") = A("k") * B("k,n");
26 
27 * Complex tensor expression
28 
29  E("m,n") = 2.0 * A("m,k") * B("k,n") + C("k,n") * D("k,m");
