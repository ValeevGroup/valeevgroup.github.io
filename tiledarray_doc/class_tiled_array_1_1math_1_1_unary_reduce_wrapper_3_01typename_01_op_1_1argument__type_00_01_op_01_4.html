Detailed Description

template<typename Op>
class TiledArray::math::UnaryReduceWrapper< typename Op::argument_type, Op >

Unary reduction wrapper class that handles lazy tile evaluation.

This class is a shallow wrapper for cases where the argument is not a lazy tile.
