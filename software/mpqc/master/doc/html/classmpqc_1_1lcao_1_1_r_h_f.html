template<typename Tile , typename Policy >
mpqc::lcao::RHF< Tile, Policy >::RHF ( const KeyVal &  kv)

KeyVal constructor for RHF

Parameters
kvthe KeyVal object; it will be queried for all keywords of AOWavefunction as well as the following additional keywords:
Keyword Type Default Description
max_iter int 30 maximum number of iterations
density_builder string "eigen_solve" type of DensityBuilder, valid values are "eigen_solve" (use ESolveDensityBuilder) and "purification" (use PurificationDensityBuilder)
localizer class none will use this object to localize the occupied orbitals; if given an empty object (or, alternatively, keyword localize is set to true), will use FosterBoysLocalizer with default parameters
localize_core bool true if false, core orbitals will not be localized.
clustered_coeffs bool false if true, will recluster coeffs in DensityBuilder; will error if true and localization_method is "boys-foster(valence)" or "rrqr(valence)"
t_cut_c real 0.0 threshold in DensityBuilder, SparsePolicy only
decompo_type string "conditioned" (cholesky_inverse, inverse_sqrt, conditioned) only valid if use ESolveDensityBuilder
s_tolerance real 1e8 S condition number threshold in DensityBuilder, valid when decompo_type is set to conditioned
