Detailed Description

TiledRange1 class defines a non-uniformly-tiled, continuous, one-dimensional range. The tiling data is constructed with and stored in an array with the format {a0, a1, a2, ...}, where 0 <= a0 < a1 < a2 < ... Each tile is defined as [a0,a1), [a1,a2), ... The number of tiles in the range will be equal to one less than the number of elements in the array.

This will construct a 1D tiled range with tile boundaries {t0, t_rest} The number of tile boundaries is n + 1, where n is the number of tiles. Tiles are defined as [t0, t1), [t1, t2), [t2, t3), ...
