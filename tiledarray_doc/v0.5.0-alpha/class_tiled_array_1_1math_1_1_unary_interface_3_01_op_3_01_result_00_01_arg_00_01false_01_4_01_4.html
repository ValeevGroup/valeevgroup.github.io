Detailed Description

template<typename Result, typename Arg, template< typename, typename, bool > class Op>
class TiledArray::math::UnaryInterface< Op< Result, Arg, false > >

Unary tile operation interface base.

This base class defines unary operations with zero or non-zero tiles, and maps arguments given to the appropriate evaluation kernel.
